# 🧠 Alocação Dinâmica de Memória em C (Array)

Este projeto demonstra o uso de **alocação dinâmica de memória** em C para manipulação de arrays utilizando `malloc` e `free`.

---

## 🎯 Objetivo

Explorar conceitos fundamentais de gerenciamento de memória em C, permitindo:

- Criar arrays com tamanho definido em tempo de execução
- Liberar memória corretamente  

---

## 🛠️ Tecnologias

- Linguagem C
- Biblioteca padrão (`stdlib.h`, `stdio.h`)

---

## 📚 Conceitos abordados

- 📌 Ponteiros  
- 📌 `malloc()` – alocação de memória  
- 📌 `free()` – liberação de memória  
- 📌 Manipulação de arrays dinâmicos  

---

## 💻 Como funciona

O programa:

1. Solicita ao usuário o tamanho do array  
2. Aloca memória dinamicamente usando `malloc`  
3. Permite inserir valores no array 
5. Exibe os valores armazenados  
6. Libera a memória com `free`  

---

## ⚠️ Boas práticas aplicadas

- ✔️ Verificação de sucesso do `malloc`  
- ✔️ Uso de `free` para evitar memory leaks  
- ✔️ Manipulação segura de ponteiros
